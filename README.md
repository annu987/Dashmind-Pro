# 🚀 DashMind AI

### Conversational Business Intelligence Dashboard Engine

DashMind AI is a next-generation conversational analytics platform that converts natural language queries into dynamic executive dashboards and strategic insights in real time.

Upload any CSV dataset and instantly generate KPIs, trends, comparisons, and intelligent visualizations — without writing SQL or using complex BI tools.

---

## ✨ Features

* 🧠 Natural Language Dashboard Generation
* 📊 Multi-Chart Executive Overview (Trend, Distribution, Ranking)
* 📈 Smart KPI Engine with Growth Metrics
* 🔎 Natural Filter Parsing (e.g., *“Show petrol cars after 2020”*)
* 📂 CSV Upload with Automatic Schema Detection
* ⚡ Offline Heuristic Analytics Engine (No External AI API Required)
* 🔁 Follow-up Query Support with Query History
* 💡 Strategic Insight Generation
* 🎨 Premium Glassmorphism SaaS UI

---

## 🏗️ Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** TailwindCSS
* **Charts:** Recharts
* **CSV Parsing:** PapaParse
* **Icons:** Lucide
* **Architecture:** Heuristic Query Engine + Dynamic Dashboard Planner

---

## ⚙️ How It Works

1. Upload a structured CSV dataset
2. DashMind detects schema automatically
3. User asks a natural language query
4. Heuristic engine maps query → chart plan
5. System generates KPIs, charts, and insights instantly

Example queries:

* `Dashboard overview`
* `Average price by fuel type`
* `Show petrol cars after 2020`
* `Top 5 expensive models`
* `Price trend over years`
